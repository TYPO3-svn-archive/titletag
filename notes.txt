

The main goal is to modify the page title, which is by default composed from
combinations of the siteTitle (coming from the root template) and the pageTitle
(coming from the page record).

The title generation constists of two parts:
  1. Extending the default title-generation of TYPO3
  2. API to trigger title modification upon certain conditions (i.e. GET vars)



Configration
------------

1. Extending the default title-generation of TYPO3

overridePagetitle      // string/stdWrap, overrides the pagetitle
forceTitle             // string/stdWrap, replaces the complete page title
pageTitleSeparator     // string/stdWrap, Only for TYPO3 < 4.7: the separator of
                       // the title parts. As of TYPO3 4.7, the value from
                       // 'config.pageTitleSeparator' will be used

2. API




plugin.tx_titletag.getVars:

    {name} => array {
        getVar => tx_some_var             // the _GET var to be checked, arrays
                                          // are splitted with |  
        check = notEmpty | int | int+     // additional check: by default, the
                                          // extension checks only, if the 
                                          // getVar exists in the request
        triggers = text | 
        content = "any string"
        content.wrap
    }
