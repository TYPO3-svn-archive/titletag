==============================
Introduction
==============================

This extension adds some more possibilities to configure the <title> tag with
typoscript.

If you are fine with the default page title generation of TYPO3, there
is no need to install this extension.

Features
==============================
* TYPO3 below 6.0: Adds stdWrap to config.noPageTitle, config.pageTitleFirst and
  config.pageTitleSeparator
* Overriding the default page title (without sitetitle) with typoscript
* Force-overriding the whole page title with typoscript
* Full compatibility with TYPO3 4.5 - 6.2 LTS
