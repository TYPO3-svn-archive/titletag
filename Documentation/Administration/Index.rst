Administration
==============================

Installation
==============================
* Import and install the extension via extension manager
* Include the static template 'titletag'


Enable extension
==============================
To get started with the configuration, you need to enable the
extension in the constants editor. The option is found in the category
'ADVANCED':

::

   config.tx_titletag.enable = 1

==============================
